# 🌍 WorldLens (Country Explorer)

## 📌 Description

WorldLens is a web-based Country Explorer application that allows users to explore detailed information about countries across the globe. The application fetches real-time data from a public API and presents it through a clean, interactive, and responsive user interface.

Users can easily search for countries, filter them by region, and sort results based on parameters such as population or name. This project focuses on applying core JavaScript concepts like API integration, array higher-order functions, and dynamic DOM manipulation.

## 🎯 Objective

The main objective of this project is to demonstrate:

* Integration of public APIs using `fetch`
* Use of array methods like `map`, `filter`, and `sort`
* Building responsive and user-friendly UI
* Writing clean, modular, and maintainable code

## 🌐 API Used

This project uses the **REST Countries API** to fetch data such as:

* Country name
* Capital
* Population
* Region
* Flag

## 🚀 Features

* 🔍 Search countries by name
* 🌎 Filter countries by region
* 🔃 Sort countries (population, alphabetical)
* 📱 Fully responsive design
* 🌙 Dark/Light mode (optional)

## 🛠️ Tech Stack

* HTML
* CSS (or Tailwind CSS)
* JavaScript (ES6)

## 📂 How to Run

1. Clone the repository
2. Open the project folder
3. Run `index.html` in your browser
